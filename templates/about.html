Criando um Website de Postagens

Exercício da disciplina "PJI110 - Projeto Integrador em Computação 1

Tecnologias usadas

  • Pyhon 3
  • Flask
  • Jinja
  • Bootstrap
  • HTML, JavaScript, CSS
  • SQLite 3
  • SQL Alchemy

Etapas das atividades

  1. Configurar um ambiente de programação
  2. Instalar as bibliotecas necessárias
  3. Criar uma aplicação base
  4. Usar templates HTML
  5. Incorporar o Bootstrap
  6. Configurar o banco de dados
  7. Mostrar todas as postagens
  8. Mostrar detalhes de uma postagem
  9. Criar uma nova postagem
  10. Editar uma postagem existente
  11. Apagar uma postagem existente

Ambiente de proramação

Importante criar um ambiente isolado de programação (env) para:

  • Evitar conflitos entre versões
  • Organizar melhor as bibliotecas necessárias utilizadas no projeto
  • Manter apenas as bibliotecas principais no diretório principal
